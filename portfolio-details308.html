Project Description

Joystream's Governance App is called "Pioneer" and it is an open-source governance application that allows users to participate in the decentralized autonomous organization (DAO) that powers the Joystream platform. With Pioneer, users can create, fund, and work on bounties, as well as run and vote for proposals. One unique feature of Pioneer is its on-chain forum, which enables decentralized discussions among Joystream users. Pioneer is part of the Joystream Monorepo on GitHub, which also includes other Joystream applications like Atlas, a whitelabel consumer and publisher experience. To access the Joystream DAO Governance App, you need to use a desktop browser and have the Polkadot browser plugin installed. Joystream is built on an open blockchain and powers multiple video streaming apps at once.
